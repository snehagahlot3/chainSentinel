ChainSentinel 

A predictive supply chain control tower that automates inventory management, alerts, and supplier ordering.

 Features

- Real-time inventory tracking
- Low-stock alerts
- Auto-order generation
- Distributor dashboard
- Supplier portal
- E-commerce integration (demo)

 Concept

ChainSentinel connects with e-commerce platforms via APIs/webhooks.
When stock decreases:

- Alerts are triggered
- Orders are automatically placed to suppliers

 Tech Stack

- Next.js
- PostgreSQL
- Prisma
- Tailwind CSS

 How it works

1. User connects store
2. Sales reduce inventory
3. System detects low stock
4. Auto-order sent to supplier
